隐私政策

最后更新日期：2025年12月24日

欢迎使用识光（ShiGuang）！我们非常重视您的隐私保护和个人信息安全。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的信息。

请您仔细阅读本隐私政策。当您使用我们的服务时，即表示您已同意本政策中描述的内容。

1. 信息收集

1.1 我们收集的信息类型

账户信息：

  • Apple ID：当您使用 Apple Sign In 登录时，我们会收集您的 Apple ID 用户标识符（非个人 Email）
  • 昵称和头像：您可以自定义昵称和上传头像，这些信息仅存储在您的设备本地

使用数据：

  • 照片：您拍摄或选择的照片用于 AI 分析。重要：这些照片仅在分析时临时发送给 AI 服务提供商，我们不会在服务器上存储您的照片原图。
  • 分析结果：AI 分析生成的知识点（标题、内容、标签等）存储在您的设备本地
  • 订阅状态：您的订阅信息（订阅类型、到期时间）由 Apple 管理，我们仅在本地记录状态

设备信息：

  • 设备型号和操作系统版本（用于技术支持和问题排查）

1.2 我们不收集的信息

  • 位置信息：我们不收集您的地理位置
  • 通讯录：我们不访问您的通讯录
  • 其他应用数据：我们不访问您设备上的其他应用数据

2. 信息使用

我们使用收集的信息用于以下目的：

  1. 提供核心服务：使用您的照片进行 AI 分析，生成知识点
  2. 账户管理：管理您的登录状态和用户资料
  3. 订阅管理：验证您的订阅状态，提供相应权限
  4. 改进服务：分析使用模式以改进应用功能和用户体验
  5. 技术支持：响应您的支持请求和解决技术问题

3. 信息共享

3.1 第三方服务提供商

为了提供服务，我们会与以下第三方共享必要的信息：

  • 豆包 AI（字节跳动）：您的照片会被发送到豆包 AI API 进行分析。豆包 AI 遵守其自身的隐私政策，我们建议您查阅其隐私条款。
  • Apple（App Store 和 StoreKit）：订阅和支付信息由 Apple 处理，我们无法访问您的支付详情。

3.2 我们不会出售您的数据

我们绝不会向任何第三方出售、出租或交易您的个人信息。

3.3 法律要求

在以下情况下，我们可能被要求披露您的信息：

  • 遵守法律法规、法院命令或政府要求
  • 保护我们的合法权益
  • 防止欺诈或安全威胁

4. 数据存储和安全

4.1 本地存储

您的所有数据主要存储在您的设备本地：

  • 知识点数据使用 SwiftData 存储在您的 iPhone 上
  • 用户资料（昵称、头像）存储在 UserDefaults 中
  • 我们不会将这些数据上传到我们的服务器

4.2 安全措施

我们采取以下措施保护您的数据：

  • 所有与第三方服务的通信均使用 HTTPS 加密
  • 敏感数据（如 API 密钥）使用安全存储
  • 定期审查和更新安全措施

4.3 数据保留

  • 照片：仅在 AI 分析时临时使用，分析完成后立即删除，不在服务器保留
  • 知识点：永久保存在您的设备上，直至您手动删除
  • 账户信息：在您删除账户后立即从本地清除

5. 您的权利

根据适用的数据保护法律（包括 GDPR 和 CCPA），您拥有以下权利：

5.1 访问权

您可以随时在应用内查看您的所有数据。

5.2 导出权

在"设置 → 数据与隐私 → 导出数据"中，您可以导出所有知识点为 JSON 或文本格式。

5.3 删除权

  • 删除单个知识点：在知识详情页删除
  • 删除所有数据：在"设置 → 数据与隐私 → 删除所有知识点"
  • 删除账户：在"设置 → 数据与隐私 → 删除账户"（将同时删除账户和所有本地数据）

5.4 撤销同意

您可以随时卸载应用以停止使用服务。卸载应用将删除所有本地存储的数据。

6. 儿童隐私

我们的服务适用于 13 岁及以上 的用户。我们不会故意收集 13 岁以下儿童的个人信息。如果您是家长或监护人，发现您的孩子向我们提供了个人信息，请联系我们，我们将立即删除相关信息。

7. 隐私政策变更

我们可能会不时更新本隐私政策。更新后的政策将在应用内和本页面发布，并标注新的"最后更新日期"。重大变更时，我们会在应用内通知您。

继续使用服务即表示您接受更新后的隐私政策。

8. 国际数据传输

您的数据可能会被传输到您所在国家/地区以外的地方进行处理（例如，发送到豆包 AI 的服务器）。我们会确保这些传输符合适用的数据保护法律。

本隐私政策受中华人民共和国法律管辖。如发生争议，双方应友好协商解决；协商不成的，任何一方可向有管辖权的人民法院提起诉讼。
